# 🌱 Carbon Emissions Reporting Platform

A full-stack web application that enables organizations to efficiently monitor, manage, and report carbon emissions across various business operations. The platform simplifies emission tracking by allowing users to record emission sources, calculate carbon footprints, visualize trends through interactive dashboards, and generate comprehensive sustainability reports.

The project aims to support businesses in making data-driven environmental decisions while promoting transparency and compliance with carbon reporting standards.

---

# 📌 Features

* 🔐 Secure User Authentication and Authorization
* 📊 Interactive Dashboard with Emission Analytics
* 🌍 Carbon Emission Data Management
* 🧮 Automatic Carbon Footprint Calculation
* 📈 Visual Reports using Charts and Graphs
* 📄 Carbon Emission Report Generation
* 🔍 Search, Filter, and Manage Records
* 📱 Responsive and User-Friendly Interface
* ⚡ RESTful API Integration
* 💾 Persistent Data Storage

# 🚀 Getting Started

## Clone the Repository

```bash
git clone https://github.com/ShriyaBanoth/Exascale-Assignments.git
```

Navigate to the project:

```bash
cd Exascale-Assignments/carbon-emissions-reporting-platform
```

---

# 📦 Install Dependencies

## Frontend

```bash
cd client
npm install
```

## Backend

```bash
cd ../server
npm install
```

---

# ▶️ Run the Application

### Start Backend

```bash
npm start
```

or

```bash
npm run dev
```

### Start Frontend

```bash
npm start
```

3. Carbon footprint is calculated automatically.
